About SleePyCat 🦾😺

SleePyCat is a simple journal application built with Flask and SQLite.

It allows users to create an account, log in, and write journal entries. The application is built with Flask, a Python web framework, and MongoDB, a non-relational database engine. The application uses JSON web tokens to manage user sessions. The application is styled with Bootstrap, a popular CSS framework.

Flask

A lightweight WSGI web application framework in Python. 👨‍🔬

MongoDB
